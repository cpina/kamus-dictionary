Information

What is this?

First and most important thing: currently this is a personal toy project. Do not rely on Kamus. It might have downtimes, might disappear, it might change the structure. On the other hand I'm using it daily so I plan to keep maintaining.

I had two things in mind:

  • How to make translation data from Wiktionary easier to access? (starting for me)
  • Learn more about Wiktionary structure and project

Kamus dictionary tries to help a user to find the translations from Wiktionary. It reads the relevant pages and presents translation information in an easier way filtering out what's not needed for the translations.

As an example, try to find the translation from English "table" to Catalan using the Wiktionary page or the Kamus page.

I hope that Kamus might help some users to use Wiktionary and hopefully to improve Wiktionary.

More languages are coming

Probably the next ones will be from Catalan and Spanish. I have some ideas to make "from any to any" instead of having to implement languages manually but I'm still evaluating it.

Quick "good to know"

When loading a translation the data is fetched from Wiktionary in real time. If you edit Wiktionary and reload Kamus you will see the improvements in real time.

The list of words for the autocompletion is fetched regularly (about once a week). New entries on Wiktionary will take some time to appear in the autocompletion list of Kamus.
